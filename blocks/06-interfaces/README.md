# Block 06 — Interfaces

## Topic
`interface` syntax, extending interfaces, declaration merging, describing function and
indexable types, and the practical `interface` vs `type` decision.

## Exercises
1. Define and extend a `Vehicle` -> `Car` hierarchy.
2. Type a `Logger` interface with a call signature.
3. Implement an object satisfying a plugin interface.
4. Use declaration merging to add a field to an interface.

Run: `npx vitest blocks/06-interfaces`
