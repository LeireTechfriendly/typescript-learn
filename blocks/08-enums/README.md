# Block 08 — Enums

## Topic
Numeric and string enums, `const enum`, reverse mappings, and when to prefer union
literals instead of enums.

## Exercises
1. Model `OrderStatus` as a string enum and map statuses to labels.
2. Convert an enum-based `switch` into an exhaustive handler.
3. Rewrite an enum as a literal union and compare trade-offs.

Run: `npx vitest blocks/08-enums`
