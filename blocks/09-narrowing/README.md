# Block 09 — Narrowing

## Topic
`typeof`, `instanceof`, `in`, truthiness and equality narrowing; user-defined type
guards (`x is Foo`); discriminated unions; exhaustiveness checking with `never`.

## Exercises
1. Narrow a `Shape` discriminated union to compute area.
2. Write an `isString` user-defined type guard.
3. Build an exhaustive reducer with a `never` default case.
4. Use `in` to distinguish two object shapes.

Run: `npx vitest blocks/09-narrowing`
