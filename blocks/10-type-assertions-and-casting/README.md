# Block 10 — Type Assertions & Casting

## Topic
`as` assertions, `as const`, the non-null assertion (`!`), double assertions, the
`satisfies` operator, and when assertions are safe vs. dangerous.

## Exercises
1. Safely parse JSON and assert a known shape.
2. Use `satisfies` to validate a config object without widening it.
3. Replace an unsafe `as any` with a proper narrowing.

Run: `npx vitest blocks/10-type-assertions-and-casting`
