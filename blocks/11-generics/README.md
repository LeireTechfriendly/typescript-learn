# Block 11 — Generics

## Topic
Generic functions, interfaces, and classes; type-parameter inference; constraints
(`<T extends ...>`); default type parameters; multiple type parameters.

## Exercises
1. `identity<T>` and `first<T>(items)`.
2. A generic `Stack<T>` class.
3. Constrained `pluck<T, K extends keyof T>`.
4. `merge<A, B>` returning `A & B`.

Run: `npx vitest blocks/11-generics`
