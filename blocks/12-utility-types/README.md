# Block 12 — Utility Types

## Topic
Built-in helpers: `Partial`, `Required`, `Readonly`, `Pick`, `Omit`, `Record`,
`ReturnType`, `Parameters`, `Awaited`, `NonNullable`.

## Exercises
1. Build an update payload type with `Partial`.
2. Create a typed lookup table with `Record`.
3. Strip fields with `Omit`; select with `Pick`.
4. Derive a function`s result type with `ReturnType`.

Run: `npx vitest blocks/12-utility-types`
