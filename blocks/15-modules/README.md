# Block 15 — Modules

## Topic
ES modules: named vs default exports, re-exports, `import type` (type-only imports), and
module resolution basics.

## Exercises
1. Split a small app into `math.ts`, `format.ts`, and `index.ts` with typed imports.
2. Use `import type` for type-only dependencies.
3. Create a barrel (`index.ts`) that re-exports a public API.

Run: `npx vitest blocks/15-modules`
