# Block 16 — Async & Promises

## Topic
Typing `Promise<T>`, `async`/`await`, `Awaited<T>`, and typing fetch-like functions and
their error paths.

## Exercises
1. Type an async `loadUser(id)` returning `Promise<User>`.
2. Sequence vs. parallelize typed promises (`Promise.all`).
3. Type a retry wrapper around an async function.

Run: `npx vitest blocks/16-async-and-promises`
