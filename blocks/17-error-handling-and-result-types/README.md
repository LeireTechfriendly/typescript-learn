# Block 17 — Error Handling & Result Types

## Topic
`try/catch` where the caught error is `unknown`, custom error classes, and the
`Result<T, E>` pattern as a typed alternative to throwing.

## Exercises
1. `tryParse(json)` returning a `Result`.
2. A `divide(a, b)` that returns an error result instead of throwing.
3. Narrow caught `unknown` errors safely.

Run: `npx vitest blocks/17-error-handling-and-result-types`
