# Block 18 — Declaration Files & @types

## Topic
What `.d.ts` files are, using `@types/*` packages, writing ambient declarations, and
typing an untyped JS module.

## Exercises
1. Write a `.d.ts` describing a small JS module.
2. Augment a global interface.
3. Add module declarations for an untyped dependency.

Run: `npx vitest blocks/18-declaration-files-and-types`
