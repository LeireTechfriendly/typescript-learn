# Block 19 — tsconfig & strict mode

## Topic
Key compiler options: `strict`, `noUncheckedIndexedAccess`, `noImplicitAny`,
`target`/`module`, and `paths`. Reading and fixing real strictness errors.

## Exercises
1. Tighten a deliberately loose config and fix the errors it surfaces.
2. Explain what each flipped flag changed (written answer in code comments).
3. Configure a path alias and import through it.

Run: `npx vitest blocks/19-tsconfig-and-strict-mode`
